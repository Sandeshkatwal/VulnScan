# Safe Authenticated Parameter Replay Planner Data

This directory stores local planning fixtures for Version 21.4.

Replay Plans and Redacted Request Templates are manual validation aids only. VulScan does not replay requests, mutate parameters, submit forms, send payloads, perform automatic authorization testing, or store raw credentials, cookies, bearer tokens, passwords, CSRF values, or session tokens.

Use Authorised Test Accounts Only and keep evidence redacted.
