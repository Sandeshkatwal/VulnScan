# Access Control Manual Test Planner Samples

These files are fake local/demo examples for Version 21.3 Access Control Manual Test Planner.

- Manual plans are documentation and workflow objects only.
- Use Authorised Test Accounts Only.
- Do not store usernames, passwords, session cookies, bearer tokens, Authorization headers, or secret authentication material.
- Do not store raw response bodies.
- Evidence file paths must stay under `reports/access_control_tests/evidence`.
- No live requests are performed by these sample workflows.

- `sample_a01_test_plan.json`
- `sample_a01_evidence_template.json`
