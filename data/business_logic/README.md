# Business Logic Review Data

This directory stores local Business Logic Review sample records.

Workflow Review Plans, State Transition Review maps, and Abuse Case Checklists are manual documentation records only. VulScan does not execute checkout, payment, approval, coupon, rate-limit, or state-changing workflows automatically.

Use Authorised Test Data Only. Do not store raw credentials, cookies, bearer tokens, passwords, CSRF values, or session tokens.
