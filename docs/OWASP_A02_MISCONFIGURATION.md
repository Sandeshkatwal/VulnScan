# OWASP A02 Security Misconfiguration

VulScan reports A02 Security Misconfiguration indicators from authorised, non-destructive evidence such as header review, endpoint classification, safe validation results, and manual evidence.

Version 20.9 reporting note: A02 evidence is consolidated into the unified OWASP Assessment report with coverage matrix status, evidence strength, confidence, and developer remediation guidance for security headers, debug exposure, and configuration hardening.
