# Screenshots

Use dashboard demo mode for clean, safe portfolio screenshots.

## Suggested Screenshot List

1. Dashboard Overview
2. Jobs page
3. Vulnerability list
4. Finding detail drawer
5. Risk overview
6. Trends view
7. Reports view
8. Remediation view
9. Settings page
10. HTML report output

## Tips

- Use demo mode.
- Use portfolio mode for presentation framing.
- Use screenshot mode for the built-in capture guide.
- Hide real paths if needed.
- Do not show secrets.
- Do not show real client data.
- Do not show real API keys, passwords, tokens, private keys, or internal hostnames.
- Use 1920x1080 or 1440p screenshots.
