## Capture Tips

- Use demo mode.
- Do not show real targets, client data, programme data, API keys, tokens, usernames, passwords, private paths, or private information.
- Keep browser zoom at 90-100%.
- Use 1920x1080 or 1440p.
- Prefer portfolio mode and screenshot mode for presentation framing.
- Check the address bar and terminal before capturing.
- Avoid screenshots that show local `.env` files, shell history, or private directories.
