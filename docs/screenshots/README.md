# Screenshot Placeholders

Use Portfolio Demo Mode and Screenshot-Ready View before capturing screenshots. Recommended filenames:

- `dashboard-home.png`
- `demo-mode.png`
- `owasp-report.png`
- `authenticated-assessment.png`
- `evidence-vault.png`
- `finding-builder.png`
- `report-composer.png`
- `business-logic-review.png`

Do not capture raw secrets, real customer data, cookies, bearer tokens, passwords, or private keys.
